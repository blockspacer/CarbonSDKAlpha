Detailed Description

Represents scene landscape data. Derived from spatial tree in order to supply hierarchical horizon occlusion culling.

Member Enumeration Documentation

enum cgLandscape::DrawResult
Enumerator:
HorizonOccluded 

Specified horizon was visible on the screen but entirely occluded.

HorizonVisible 

Horizon was visible above the current horizon line.

HorizonClipped 

The horizon geometry was off the screen and could not be tested.

HorizonInvalid 

The horizon edge was invalid (perhaps it ran in the incorrect direction).
