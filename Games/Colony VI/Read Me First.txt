General Controls
----------------
Pause & Menu       = Escape


Movement Controls
-----------------
Forward            = W
Backward           = S
Strafe Left        = A
Strafe Right       = D
Walk               = Shift
Crouch             = Control
Jump               = Space


Weapon Controls
---------------
Fire               = Left Mouse Button
Aim                = Right Mouse Button
Reload             = R
Change Fire Mode   = X
Throw Grenade      = G
Select Pistol      = 1
Select Rifle       = 2
Select RPG         = 3 (When collected)
