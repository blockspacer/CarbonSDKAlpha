cgKinematicControllerJointObject Class Reference

Class implementing a simple kinematic controller joint as a scene object. This joint can be used to anchor a body to a specific point in space. This anchor point can then be manipulated dynamically at runtime resulting in the body following. More...

Detailed Description

Class implementing a simple kinematic controller joint as a scene object. This joint can be used to anchor a body to a specific point in space. This anchor point can then be manipulated dynamically at runtime resulting in the body following.
