void setCellLayout (const cgPointF &cellOffset, const cgSizeF &cellSize, const cgSize &cellGridLayout)
 Clutter cells can house more than one displacement / layer map in a grid like pattern. For instance, one clutter cell might house data for a collection of 2x2 terrain blocks, or any other subdivided section of a larger world region. This method can be called to alter the internal grid grid from its default of 1x1, to any larger grid layout, as well as to supply cell offset and size data. This method must be called prior to applying displacement or layer maps.

 Supply the displacement map for a specific grid square within this cell. This is used to compute the correct Y axis offset for each clutter element. Note: no copy will be made of the displacement data, and thus the lifetime of this information must be guaranteed.
