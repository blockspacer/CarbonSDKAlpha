Detailed Description

When rendering sections of the object render queue, render control script side "techniques" can be supplied that customize the necessary rendering behaviors. The context class is provided to these 'techniques' as a means of describing the objects currently being processed, as well as supplying functionality to operate on them (i.e. culling, rendering, etc.)
