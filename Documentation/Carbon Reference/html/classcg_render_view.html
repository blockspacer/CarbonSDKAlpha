cgRenderView Class Reference

A render view describes an area of the main frame buffer into which the application should render. A little more comprehensive than a simple 'viewport', this class essentially provides a rendering 'sandbox' with unique rendering surfaces that fit precisely within the view. More...

Detailed Description

A render view describes an area of the main frame buffer into which the application should render. A little more comprehensive than a simple 'viewport', this class essentially provides a rendering 'sandbox' with unique rendering surfaces that fit precisely within the view.
